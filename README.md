# Telegram Parody Bot

Бот для групповых чатов, который передразнивает сообщения: сначала по правилам замены слов, при неудаче — через OpenAI.

## Пример

- Сообщение: `мне вчера было грустно`
- Ответ бота: `хуюстно`

## Настройка Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите `BOT_TOKEN`.
2. Добавьте бота в группу.
3. Отключите Privacy Mode: `/setprivacy` → выберите бота → **Disable**.  
   Иначе бот не увидит обычные сообщения и не сможет отвечать случайно.
4. Убедитесь, что бот может отправлять сообщения в группе.

## Поведение

- Отвечает **всегда**, если на него ответили reply.
- На обычные сообщения отвечает с настраиваемой вероятностью (по умолчанию из `.env`).
- В чате: `/chance` — шанс для всех, `/userchance @ник 50` — для участника, `/cooldown` — пауза.
- Команды может писать любой участник чата.
- Работает только с текстом в group/supergroup.
- Ответ отправляется reply на исходное сообщение.

## Как передразнивает

Бот берёт **последнее слово** и отвечает **одним словом** по правилам:

- **-но** → `хуйно`; если перед `-но` есть **у** → `хую` + хвост (`грустно` → `хуюстно`); **асно** → `хуясно`
- **-то / -да** → ищет гласную перед окончанием; **у** → `хую` (`круто` → `хуюто`), **а** → `хуя`
- префикс подбирается по гласной в основе: `о`→хуй, `и`→хуи, `е`→хуе и т.д.

| Слово | Ответ |
|-------|-------|
| грустно | хуюстно |
| круто | хуюто |
| просто | хуйосто |
| чисто | хуисто |
| лето | хуето |
| правда | хуявда |
| опасно | хуясно |
| пиво | хуиво |
| сообщение | хуение |
| программа | хуямма |
| капец | хуец |
| монитор | хуитор |

Примеры — в [`rules/examples.json`](rules/examples.json).
